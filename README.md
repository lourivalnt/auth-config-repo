# Auth Config Repository

Repositório central de configurações da Auth API.

## Estrutura

- auth-api.yml → configuração base
- auth-api-docker.yml → ambiente Docker
- auth-api-prod.yml → produção

## Responsabilidades

- Feature flags
- Configuração de segurança
- Parâmetros de observabilidade

⚠️ Este repositório não contém código.
